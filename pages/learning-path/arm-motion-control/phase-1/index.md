# 第一阶段 - 理论与编程基础

本模块是机器人工程的核心支柱。涵盖了从描述物理世界的数学工具——**机械臂运动学与动力学**，实现控制逻辑的工程手段（**C++/Python**）到算法验证的科学计算平台（**MATLAB**）

> **学习建议：** 理论学习切忌求全。在掌握坐标变换（DH 参数/指数积）、基础编程语法及控制系统传递函数后，应立即进入工程实践。深刻的理论理解往往源于代码报错时的调试与物理仿真中的偏差校正
