# 关于我

你好，我是 **Kaede Rei**。

目前是 **华南农业大学本科生**，同时参与 **广东省农业科学院** 相关项目协作。  
我主要围绕 **嵌入式系统、六轴机械臂、ROS 系统集成与具身智能** 展开学习与工程实践，聚焦农业机器人场景下从底层驱动到上层智能控制的完整技术链路。

## 我在做什么

我长期关注农业机器人相关系统的工程落地，尤其是：

- 从底层驱动到运动控制的完整实现
- 机械臂电控系统、运动学求解与轨迹规划
- 控制、视觉、导航三部分的 ROS 系统集成
- IL / RL 策略在真实机械臂平台上的部署与验证

## 我的关注重点

相比只做“能跑起来”，我更关注这些问题：

- **可靠性**：系统是否能长期稳定运行，而不是一次性演示成功
- **实时性**：控制链路是否具备明确的时序与边界
- **工程清晰度**：模块职责、通信接口、状态流转是否足够清楚
- **场景适应性**：系统是否能在非结构化农业环境中保持稳定表现

## 代表项目

### 番茄采摘机器人
围绕 **STM32 + ROS + MoveIt** 构建机械臂底层驱动、运动学求解与 ROS 控制链路，面向精准农业场景进行工程落地。

### 鸽笼自动投喂系统
将系统从裸机架构重构到 **RTOS**，提升系统稳定性、可维护性与任务调度能力。

### 导轨转向底盘
负责底盘转向控制、底盘运动学与通信协议设计，面向农业移动平台场景进行开发与验证。

### 双臂 IL + RL 系统
基于 **LeRobot + ROS2**，探索模仿学习与强化学习在真实机械臂平台上的协同部署。
